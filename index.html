Predicting Cancer Morbidity using State Space Model

Here I explore cancer data from SEER and CDC WONDER websites using SAS. The purpose of this project was to see if the State Space Model was an appropriate model to predict cancer. It is one of many Time Series Analysis and uses previous states of information to model the current state. Once modeled you then must smooth the data in order to predict future observations. My github page has files containing the final project as well as the SAS code used to perform these predictions.
